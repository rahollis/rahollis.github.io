def plot_bw(pic_file_path):
    img_arr = plt.imread(pic_file_path)
    img_arr = img_arr[:,:,0]
    plt.imshow(img_arr,cmap='gray')

plot_bw('boston.png')

Notes¶

plt.imread() is the function that grabs a picture file and converts it into a numpy array.

  • .png files are typically stored as 3D arrays of floats between 0 and 1 (4 sheets).
  • .jpg files are typically stored as 3D arrays of integers between 0 and 255 (3 sheets).
    • Check by using np.max()
