
Exam 1 Rubric
=============

Each question will be graded out of 4 points, with partial credit
given for incomplete or incorrect answers.  To pass the exam, you
will need at least 10 points total, which is an average of 2.5
for each question.

Next to each part of the rubrics, you will see error codes in
square brackets.  These error codes are will be used to indicate
why any points were taken off on your exam papers.


Question 1
~~~~~~~~~~

*Write a system of equations for the given application.*

(2 pts) All equations are present
	(-1 pt) Missing an equation or two [ME]

	(-2 pts) Missing 3 or more equations [ME]
(1 pt) All equations are in standard form
	(-1 pt) Equations are not in standard form [SF]
(1 pt) All equations are correct
	(-1 pt) 3 or more equations have mistakes [IE]


Question 2
~~~~~~~~~~

*Solve the system of equations.*

(1 pt) All work included
	(-1 pt) Most of the work is not included [MW]
(1 pt) Correct solution
	(-1 pt) Solution is incorrect [IS]
(2 pts) Problem is solved correctly
	(-2 pts) Solution is not used to answer the question [MA]

	(-1 pt) Answer is not correct [IA]


Question 3
~~~~~~~~~~

*True or False?*

(1 pt) Correct answer
	(-1 pt) Answer is either incorrect or missing [IA]
(3 pts) Explain answer
	Answer TRUE
		Provided reasoning (3 points possible)
			(-1 pt) Reasoning is sound but not a proof [SR]

			(-3 pts) Reasoning is not sound [BS]
		Provided an example (2 points possible)
			(-2 pts) Example supports an answer of TRUE [EX]

			(-1 pt) Example supports an answer of FALSE (is a counterexample) [CX]
	Answer FALSE
		Provided reasoning (3 points possible)
			(-2 pts) Reasoning is sound but not a proof of the negative [SR]

			(-3 pts) Reasoning is not sound [BS]
		Provided an example (3 points possible)
			(-2 pts) Example supports an answer of TRUE [EX]

Question 4
~~~~~~~~~~

*How many solutions?*

(1 pt) Compute RREF
	(-1 pt) Computed RREF is incorrect or missing [CE]
(1 pt) Identify pivot columns
	(-1 pt) Incorrectly identify pivots [IP]
(2 pts) Use number and locations of pivot columns to determine number of solutions
	(-1 pt) Answer is incomplete or has a minor error [ME]

	(-2 pts) Answer is incorrect or missing [IA]
